# Tarea1InterfazGrafica-LilliannaCepedaCandelaria

**Nombre**
LilliannaCepedaCandelaria

**Matricula**
10005856

**Materia**
Programacion 3

**Descripción**
Aplicacion 1: Registro de visitantes. El programa debe mostrar una interfaz grafica en el que se pudiera registrar estudiantes con sus datos usando campos y otras herrmaientas. Y además mostrar una lista de visitantes registrados. 
Aplicación 2: Registro de productos. El programa debe mostrar una interfaz grafica en la que se pudiera registrar los productos a partir de las caracteristicas de los productos, como nombre del producto, codigo, cantidad etc. Y además debe mostrar una lista de los productos registrados
Aplicación 3: Registro de estudiantes. El programa debe mostrar una interfaz grafica de registro de un estudiante en la que se puede agregar una foto del estudiante además de los datos del mismo. Y también debe mostrar una lista de los estudiantes registrados.
